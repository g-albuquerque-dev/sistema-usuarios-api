# 🚀 Sistema de Gerenciamento de Usuários (Full-Stack)

Bem-vindo ao repositório do Sistema de Gerenciamento de Usuários!
Este é um projeto Full-Stack em desenvolvimento contínuo, focado em aplicar as melhores práticas de Engenharia de Software, criação de APIs RESTful e segurança de dados.

## 🎯 Objetivo do Projeto
O objetivo principal desta aplicação é construir um CRUD (Create, Read, Update, Delete) completo e robusto para gerenciar o cadastro de usuários, servindo como uma peça central do meu portfólio de desenvolvimento.

## 🛠️ Tecnologias Utilizadas

### Back-end
* **Java 17** (ou 21)
* **Spring Boot** (Framework principal)
* **Spring Web** (Criação de Endpoints e API REST)
* **Spring Data JPA** (Persistência de dados e ORM)
* **Lombok** (Redução de código boilerplate)
* **Banco de Dados H2** (Banco em memória para testes ágeis)
* **Spring Security & BCrypt** *(Em breve - para criptografia e proteção)*

### Front-end
* **Angular** *(Em breve - para consumo da API e interface interativa)*
* **TypeScript** *(Em breve)*
